# 🌍 GeoGuesser — Многопользовательская игра

Веб-версия игры GeoGuesser с мультиплеером, авторизацией и статистикой.

## 🚀 Возможности

- 🎮 **Многопользовательская игра** по сети через Socket.IO
- 👤 **Авторизация** с сохранением статистики
- 🗺️ **Google Maps / Yandex Maps** панорамы
- 📊 **Подробная статистика** и достижения
- 🏆 **Рейтинг** игроков
- 📍 **Скрытые метки** — видны только после ответов всех игроков
- 📱 **PWA** — установка на телефон

## 🎮 Режимы игры

| Режим | Описание |
|-------|----------|
| 🎯 Classic | Все играют, побеждает лучший |
| 🧑 Solo | Одиночная игра |
| ⚔️ Дуэль | 1 на 1 |
| 👑 Battle Royale | Худший вылетает каждый раунд |
| 👥 Команды | 2v2, 3v3 |
| ⚡ Блиц | 15 секунд на ответ |

## ⚙️ Настройки

- Количество раундов (1-20)
- Время на раунд (10-600 сек)
- Разрешение перемещения
- Зум и компас
- Выбор карт

## 🛠 Установка

### Требования
- Node.js 18+
- MongoDB

### Запуск

```bash
# Клонировать
git clone https://github.com/your-username/geo-guesser.git
cd geo-guesser

# Сервер
cd server
cp .env.example .env  # Заполнить переменные
npm install
npm run dev

# Клиент (в другом терминале)
cd client
npm install
npm run dev
